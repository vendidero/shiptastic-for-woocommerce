# Security Policy

## Supported Versions

Generally, only the latest version of Shiptastic for WooCommerce has continued support. If a critical vulnerability is found in the current version of Shiptastic for WooCommerce, we may opt to backport any patches to previous versions.

## Reporting a Vulnerability

Shiptastic for WooCommerce is an open-source plugin for WordPress. Please report security bugs found in the source code of the plugin through the [Patchstack Vulnerability Disclosure Program](https://patchstack.com/database/vdp/6fc53392-d89b-41f3-bc28-eb3680901b4d). The Patchstack team will assist you with verification, CVE assignment, and notify the developers of this plugin.

Our most critical targets are:

Shiptastic for WooCommerce (this repository)
vendidero.de, vendidero.com -- the primary marketplace and marketing site, and all of it subdomains, e.g. download.vendidero.de
